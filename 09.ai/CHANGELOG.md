# 09.ai Changelog

本檔案記錄 `tech-notes` 專案治理架構的重大異動與進化歷程。

---

## [1.0.0] - 2026-04-19

### ✨ Added
- **正式確立 AI 治理架構**：建立以「五大核心角色」為基礎的數字公司模型。
  - `11. Legal Advisor` (法律顧問)：管憲法與記憶。
  - `12. Specialized Expert` (領域專家)：管技術 SOP。
  - `13. Design System` (視覺總監)：管 UI/UX 規範。
  - `14. Conductor` (總經理)：驅動開發進度。
  - `15. TDD Quality Manager` (品管主管)：檢查正確性。
- **Mermaid 視覺化**：在 `00.專案架構簡介.md` 加入專業與輕量級開發路徑圖。
- **雙軌制度**：區分適合小型專案的「輕量級」與適合複雜系統的「專業級」。
- **實體化 Governance Files**：初始化 `CHANGELOG.md`, `GUIDELINES.md`, `DESIGN.md`, `CUSTOM_SKILL.md`, `TEST_GUIDELINES.md`。

### 🔄 Changed
- 將原有的編號 `02.動態流程` 與 `03.法律規範` 重新編排為專業級導向的 `11-15` 系列。
- 統一 Markdown 檔案格式，加入 GitHub Alerts 與多級目錄。

### 🔧 Fixed
- 修復 Mermaid 圖表語法錯誤（包括節點 ID 含點號、平行四邊形含括號等問題）。
